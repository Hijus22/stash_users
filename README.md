# stash_users
Command for managing stash users

How to use:

```
usage: stash_users [--help|-h] [--verbose|-v] [--simulation|-s] --api|-a HOST --token|-t TOKEN --log|-l LOGPATH

  -a string
                The host url.
                        Example: https://host.com(shorthand)
  -api string
                The host url.
                        Example: https://host.com
  -h            Show this help message and exit(shorthand)
  -help
                Show this help message and exit
  -l string
                The log path where the loggers will write to.
                        Example: /var/etc/log/stash(shorthand)
  -log string
                The log path where the loggers will write to.
                        Example: /var/etc/log/stash
  -s            Execute in simulation mode(shorthand) (default true)
  -simulation
                Execute in simulation mode (default true)
  -t string
                The token needed for the Crowd Api.
                        Example: 'Basic xfg23gfjsf2=='(shorthand)
  -token string
                The token needed for the Crowd Api.
                        Example: 'Basic xfg23gfjsf2=='
  -v            Execute in verbose mode(shorthand)
  -verbose
                Execute in verbose mode
```
